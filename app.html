App

A App foi desenvolvida durante os 3 meses que estive a trabalhar na EzConnector. A EzConnector geria a base de dados das empresas que trabalhavam com ela. Já tinha um website para cada uma delas, então precisava de uma App para complementar. Então comecei esse projeto, era me dado os dados das empresas e eu só precisava de desenvolver uma App que se adapta-se aos dados que vinham da base de dados no formato json.

A App permitia visualizar os produtos, ter um carrinho de compras quando o login estivesse efetuado e depois enviar esse pedido para a base de dados e ser processado pela empresa. Tanto podia ser usada para gestão de stocks dentro da empresa (assim os empregados podiam usar uma aplicação no telemóvel) ou ser usada mesmo para E-commerce.

A linguagem utilizada foi o Dart, na framework Flutter e trabalhava com o postman para fazer os testes de conexão á base de dados. O projeto atualmente está a ser desenvolvido por outro programador, uma vez que eu tive de continuar a licenciatura e o projeto precisava de avançar mais rapidamente.
