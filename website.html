WebSite

Este Website foi desenvolvido na cadeira de PDI, o objetivo era servir qualquer canil que necessitasse de adotar um website, ele permite adicionar todos os dados há base de dados e ser gerido por um administrador.

Existem 3 tipos de utilizadores, o utilizador que não efetua login e apenas visualiza o website, o que efetua login e poder enviar um pedido para se tornar voluntário ou para adotar um cão e o administrador que consegue gerir todo o website. O administrador pode aceitar ou não o voluntário e o pedido de adoção, adicionar novos cães e gerir as notícias ou o material que o canil precisa.

A linguagem utilizada foi PHP com a framework Laravel e o servidor foi o Laragon. Também foi utilizado o Bootstrap para facilitar na programação. Este trabalho foi realizado por 3 elementos, a parte que eu desenvolvia mais era o back-end, só na parte de administrador é que desenvolvi tudo. Os outros dois elementos eram responsáveis pelo front-end do website.
